সমবায় সমিতি সমূহ

এক নজরে এই বিভাগের সকল জেলার সমবায় সমিতি সমূহ


ক্রমিকসমিতির নামসমিতির সংগঠকের নামসমিতির নিবন্ধন নং ও তারিখবিস্তারিত
0 ১ সিংগাইর বহুমূখী সমবায় সমিতি লিঃ। জনাব মোঃ এমদাদুল হক নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ০১ তারিখ- ২৮/০২/০৮ ইং। বিস্তারিত
1 ২ আপন গৃহ বহুমুখী সমবায় সমিতি লিঃ। জনাব কাজী মোঃ জিয়া উদ্দিন নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১৪, তারিখ- ২৩/০৯/১০ইং। বিস্তারিত
2 ৩ রুরাল ডেভলপমেন্ট প্রোগ্রাম ফাইন্যান্স এন্ড ইনভেস্টমেন্ট মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লিঃ। জনাব মোঃ বিল্লাল হোসেন নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ৩১৮ তারিখ- ১১/১০/০৭ ইং। বিস্তারিত
3 ৪ আল আকছা ইসলামী মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লি:। জনাব মোঃ সাইদুর রহমান নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১৭৯ তারিখ- ০৩/০৪/০৫ ইং। বিস্তারিত
4 ৫ আরবান ফাইন্যান্স এন্ড কমার্স মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লি:। জনাব খন্দকার আবুল কালাম আজাদ নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১২২২, তারিখ- ০৬/০৯/০৬ ইং। বিস্তারিত
5 ৬ সুমাইয়া মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লি:। জনাব শেখ মিরাজুল ইসলাম নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১৯২ তারিখ- ০৭/০৪/০৫ ইং। বিস্তারিত
6 ৭ আল-ফাতাহ্ মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লিঃ। জনাব মোঃ শাহিনুজ্জামান নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ০১ তারিখ- ২২/০২/০৯ ইং। বিস্তারিত
7 ৮ সেফটি ফাষ্ট মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লিঃ মোঃ মিজানুর রহমান নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১০২৫ তারিখ- ২৯/১২/০৫ ইং। বিস্তারিত
8 ৯ তামাদ্দুন মাল্টিপারপাস কো-অপারেটিভ সোসাইটি লিঃ। জনাব মোঃ জালালুদ্দীন আহমদ নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ১১৪৩ তারিখ- ২০/০৮/০৭ ইং। বিস্তারিত
9 ১০ গড়বাড়ী বাজার ক্ষুদ্র ব্যবসায়ী বহুমূখী সমবায় সমিতি লিঃ। জনাব কবির হোসেন নিবন্ধন নং- 0 তারিখ- নিবন্ধন নং- ৬২, তারিখ-২৭/০৮/০১ইং। বিস্তারিত
